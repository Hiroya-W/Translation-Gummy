Translation-Gummy is a magical gadget which enables user to be able to speak and understand other languages. You can get the translated journal pdf link by using the following command:

@gummy <journal_url> [-T <translator>]
  • journal_url : URL of a page you want to create a pdf.
  • translator : translator identifier, string name of a translator.

NOTE:

This Slack App does not send or receive PDF, so it does not overwhelm the slack capacity :)
